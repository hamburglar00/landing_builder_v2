## Migraciones de Supabase

Este directorio está reservado para las migraciones SQL del proyecto.

- Aquí iremos agregando los archivos `.sql` que definan el esquema de base de datos (usuarios, perfiles, clientes, etc.).
- Más adelante podemos usar `supabase db diff` y `supabase db push` para sincronizar el esquema entre local y el proyecto remoto.
